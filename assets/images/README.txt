Instructions for adding a profile picture:

1. Place a square image file named 'profile_pic.jpg' in this directory
2. The image should be at least 120x120 pixels for best quality
3. Supported formats: jpg, png
4. If no image is provided, the app will show an error or fallback

You can download sample profile pictures from:
https://drive.google.com/drive/folders/1I4_qxFgwpNOa8f_66lJG9Mzig4XbATr8?usp=share_link

Make sure to name your image exactly: profile_pic.jpg
